This project analyzes the adoption of Hybrid Maize Seeds (HMS) among maize farmers in Nigeria, using survey data from 1,000 respondents across four states. The study investigates how socio-economic, agronomic, and institutional factors influence adoption decisions and forecasts future adoption trends.

The analysis supports agricultural policy design, agribusiness strategy, and development research.
